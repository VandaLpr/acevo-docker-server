# AC EVO Docker Server

Run Assetto Corsa EVO Dedicated Server in Docker using Wine.

## Features
- Wine-based server
- Auto restart
- Headless (no GUI)
- Docker / Unraid compatible

## Usage

1. Copy your server files into `serverfiles/`
2. Edit SERVER_ARGS
3. Run:

docker compose up -d

## Ports
- 9700 TCP
- 9700 UDP
